APPENDIX C
Site Photographs
This Volume No 4 book is provided as a supplemental reference for the
Rockslide Potential Inspections project for the City and County of Honolulu This book
contains selected photographs that depict the general site conditions for the hillslopes
that were evaluated for this preliminary hillslope inventory and rating study In addition
to the hard copy presentation of the photographs the electronic files for the
photographs contained in this book are also provided
Site Photographs
Digital photographs were recorded for each hillslope site evaluated for the
rockslide potential study The recorded photographs which were taken during the field
reconnaissance were reviewed and two images which best show the existing site
conditions were selected for inclusion in this report In general the selected
photographic images include both a profile view and a frontal or oblique view
Some site conditions that were occasionally encountered such as_ thick
vegetation growth and poor lighting shadows and poor contrast of slope features
limited the number of good quality photographs that could be recorded Therefore
some hillslope sites do not have profile view images suitable for report reproduction
The photographs were recorded with a 1 mega pixel digital
camera The photographic images were recorded as JPEG image files For this report
the selected photographic images have been formatted into a Microsoft Word 2000
document The Word 2000 electronic file for the photographs in Appendix C is
provided as part of this report
